Instructions

  1. Click on the map to set the start and end destination. The start is marked with an (S) on the map, while the destination is marked with a flag. The form on the left hand side will automatically fill when selecting the start and destination markers.

  2. Select your chosen path finding algorithm from the dropdown.

  3. Click Submit and you will be redirected to a page which shows:

    • Shortest Path Calculated Using the Algorithm
    • Visualizer for The Algorithm
    • Comparison of Path To Google Directions API Path
